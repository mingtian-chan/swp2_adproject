클래스 메서드 입력인자 출력인자 기능
MainWidget init_ui - - 메인화면
rotate_character - - 1초마다 캐릭터 움직임
ui_tick - - gamestate의 tick 함수가 주기적으로 호출될수 있게끔 ui에서 연결
update_labels_and_bars - - 사용자의 현재 레벨, 경험치, 체력, 포만감, 청결, 졸림 수치등을 업데이트
game_clicked - - 미니게임 화면으로 전환
keyPressEvent event - esc가 눌렸을 때 꺼짐
closeEvent event - writeSaveFile()호출
readTamago - - 파일에 저장된 GameState를 읽어 GameState를 생성함
writeSaveFile - - GameState관련 수치를 파일로 저장함
write_highscore - - 게임 종료시 이름과 레벨을 저장
RPSGameWidget init_ui - - 미니게임 화면
closeEvent a0 - exit_clicked() 호출
exit_clicked - - 게임종료 버튼 누르면 미니게임 종료되고 미니게임 창 닫힘
keyPressEvent event - esc가 눌렸을 때 꺼짐
play_rps player_choice - 사용자의 가위바위보 선택을 받아 승자를 판정하여 적절한 경험치를 수여함
DifficultyView create_widgets - - 난이도 창의 기본 위젯을 초기화하고 화면에 표시함
plot difficulty - 난이도 계수를 받아 plot위젯을 업데이트함
get_scoreboard - - 파일에 저장된 순위 정보를 읽어와 리스트를 반환
changeValue - - 난이도 계수를 받아와 난이도 모형을 수정후, plot()를 호출함
closeEvent a0 - 창 종료 요청시 종료 및 MainView로 focus 이양
GameState __init__ name, experience, satiety, hygiene, drowsiness, hp - GameState 게임 난이도 조절 관리자를 주어진 게임 상태값으로 생성함
get_tick - - 동적 난이도를 위해 레벨에 비례하여 게임의 목표 진행속도를 계산하요 반환함
increment_xp amount - amount만큼 사용자의 경험치를 증가함.
eat - - 포만감을 난이도 모형에 따라 증가시키고, 적분모형에 따라 적잘한 양의 경험치를 지급함
satiety_loss_func x y 포만감 수치 x에서의 단위 시간당 포만감 감소 수치 y를 반환함
drowsiness_loss_func x y 졸림 수치 x에서의 단위 시간당 졸림 감소 수치 y를 반환함
eat_gain_func x, difficulty y 난이도 계수 difficulty, 포만감 수치 x에서 먹이를 줬을 경우 포만감 증가량 y를 반황함
satiety_loss - - 현재 포만감에 대해 satiety_loss_func()를 호출하여 포만감을 차감시키고, min(0, max(100, satiety))수행함
sleep - 졸림 수치를 100으로 설정함
wash - - 청결 수치를 100으로 설정함
drowsiness_loss x y 졸림 수치 x에서의 단위 시간당 졸림 수치 감소량 y를 계산함
hygiene_loss x y 청결 수치 x에서의 단위 시간당 청결 수치 감소량 y를 계산함
hp_loss_func x y 현재 게임 수치 array x를 받아 체력 손실함수를 계산하여, 단위 시간당 체력 감소량 y를 계산함
tick - - 단위 시간에 대해 게임 시간을 1만큼 증가시키고, 게임 루프를 1회 수행함
